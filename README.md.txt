THE_JETSONS_BOOTCAMP

Introdução

Este bootcamp é um treinamento prático para aprender fundamentos de segurança cibernética usando o CMD do Windows.

Você vai aprender a:

- Consultar IP local e público
- Usar VPN e identificar mudanças de IP
- Navegar pela rede Tor e entender anonimato
- Gerar hashes de arquivos para verificar integridade
- Identificar técnicas básicas de phishing

Como usar

1. Abra o arquivo `index.html` no navegador para iniciar.
2. Identifique-se com seu número de aluno.
3. Use o menu para navegar pelas atividades.
4. Cada atividade possui comandos para rodar no Prompt de Comando do Windows.
5. Salve os resultados dos comandos nos arquivos txt indicados na sua área de trabalho.
6. Envie os arquivos txt para avaliação.

Dicas

- Sempre verifique a origem dos links.
- Use VPN para proteger seu IP real.
- Utilize Tor para maior anonimato.
- Verifique hashes para garantir que arquivos não foram alterados.
- Fique atento a e-mails e mensagens suspeitas para evitar phishing.
