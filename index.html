     BEM-VINDO AO THE_JETSONS BOOTCAMP DE SEGURANÇA - WINDOWS EDITION

    Aqui você irá aprender, passo a passo, como proteger sua
    privacidade e entender conceitos básicos de segurança.
    Prepare-se para realizar atividades práticas diretamente no CMD,
    entender seus IPs e conexões, usar VPN e Tor, extrair hashes e
    identificar phishing com atenção e cuidado.

    Complete as etapas, preencha os campos abaixo e salve seu progresso
    para que o instrutor possa avaliar seu aprendizado.

    Use as dicas para descobrir os comandos executar no CMD.
Dicas para usar o Prompt de Comando (CMD):
  • Após ativar sua VPN, use o mesmo comando para notar a diferença no endereço IP.
  • Para saber o IP via Tor, conecte ao Tor e rode o mesmo comando anterior para conferir.
  • Quer extrair o hash SHA256 de um arquivo? Pense em um comando do Windows que calcula hashes e precisa do caminho do arquivo.
  • Para identificar phishing, observe sempre links estranhos, erros ortográficos e solicitações urgentes.
