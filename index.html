Bem-vindo ao Bootcamp de Segurança - Windows Edition

Introdução

Este bootcamp vai te ensinar conceitos fundamentais de segurança usando o CMD do Windows. Você vai aprender a descobrir seu IP, usar VPN, identificar phishing e calcular hash de arquivos. Tudo passo a passo, para que você fique mais seguro no dia a dia.

Por favor, comece se identificando com seu número de aluno abaixo para prosseguir.

Atividade 1 - Descubra seu IP

No CMD, digite: ipconfig e anote o IPv4.

Dica: O IPv4 geralmente começa com 192.168.x.x ou 10.x.x.x em redes locais.

Atividade 2 - IP com VPN Proton

Ative o ProtonVPN no Windows e, no CMD, digite ipconfig novamente. Anote o novo IPv4 e compare com o anterior.

Dica: Se a VPN estiver funcionando, o IP deverá mudar para um diferente da sua rede local.

Atividade 3 - IP com Tor

Utilize o Tor Browser ou o Tor via proxy configurado no Windows. No CMD, digite curl https://check.torproject.org/api/ip para verificar seu IP Tor (você precisa ter o curl instalado, pode usar o PowerShell).

Dica: O IP do Tor será diferente do IP da sua rede local e VPN.

Atividade 4 - Hash de arquivo

No CMD, calcule o hash SHA256 de um arquivo qualquer com o comando:

CertUtil -hashfile caminho\para\arquivo SHA256
